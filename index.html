Getting Started with Mongodb

MongoDb:

  • MongoDb is a document database designed for ease of development and scaling
  • It is intitutive and easy to use NOSQL database
  • It is available as community and enterprise edition

Mongo Vs Mongod

Mongo:

  • Mongo is the command-line shell that connects to a specific instance of mongod
  • In Mongo we give only command/instruction to perform tasks and In actual mongod perform those tasks

Mongod:

  • Mongod is the "mongo Daemon " its basically host process for the database
  • Think Mongod is like a compiler
  • It is actual process which perform work to store data in database

Comparison of mongodb with mysql :

       SQL Terms           MongoDb Terms

       Database------------Database

       Tables--------------Collections

       Rows----------------Documents(BSON)

       Columns-------------Fields

Some Key Points :

  • Collection : Collections is like a JavaScript Objects
  • Document : Document is one unit of data inside collections
  • Internally It uses BSON which is much similar like JSON
  • Also BSON is more efficient than JSON

    COMMAND RELATED TO COLLECTIONS show collections = To view all collections db.createCollection('name') => To create new collection db.collectionName.drop() => To delete collection

    Commands Related to Database :

    • show dbs : to view all Database
    • use databaseName : to create a new Database
    • use databaseName : switch to particular Database name
    • db : to view current Database
    • db.dropDatabase() : to Delete Database

    Commands Related to Collections :

    • show collections : to view all collections
    • db.createCollection('name') : to create a new collection
    • db.collectionName.drop() : to delete collection
    db.comments.find() => To see all inserted rows in a COLLECTIONS db.comments.find().pretty() =>> To see all inserted rows in a COLLECTIONS(beautified)

    Commands Related to Rows :

    • db.collectionName.insert({ 'name' : 'manish', 'age' : 20, 'address' : 'patna' }) : to inset in a row
    • db.comments.insertMany([{ 'name' : 'manish', 'age' : 20, 'address':'sheohar' }, { 'name' : 'abhijeet', 'age' : 7620, 'address':'patna' }, ]) : to Insert Many Rows
    • db.comments.find() :to see all inserted rows of a collection
    • db.comments.find().pretty() :to see all inserted rows in a collections in beautified
